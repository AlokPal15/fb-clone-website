# 📘 Facebook Clone Website

## 📖 Project Overview

This project is a **Facebook Clone Website** developed using **HTML** and **CSS**. It was created as part of my web development learning journey while following a Git and GitHub tutorial. The main objective of this project is to understand the fundamentals of front-end web development and practice using Git for version control.

The website replicates the basic user interface of Facebook, including its homepage layout, navigation bar, sidebar, news feed section, and user profile elements. Although this is not a fully functional social media platform, it closely resembles Facebook's design and helps in improving HTML, CSS, and Git skills.

---

- Responsive Facebook-inspired homepage
- Navigation bar with icons and search box
